# Astra Launcher

> **A minimal, productivity-first Android launcher built for students
> and developers.**

Astra Launcher transforms the Android home screen into a focused
workspace instead of a traditional app grid. It combines a clean
monochrome interface with productivity tools such as tasks, notes,
calendar integration, unified search, and a distraction-free dashboard.

------------------------------------------------------------------------

## ✨ Features

### Week 1

-   Minimal Material 3 UI built with Jetpack Compose
-   Large productivity-focused home screen
-   Floating dock
-   App drawer with search
-   Settings with DataStore
-   MVVM + Clean Architecture
-   Hilt Dependency Injection

### Week 2

-   Task Manager
    -   Create, edit, delete tasks
    -   Priority levels
    -   Due date & time
    -   Completion tracking
-   Notes
    -   Quick notes
    -   Pin notes
    -   Edit & delete
-   Calendar integration
-   Unified search across:
    -   Apps
    -   Tasks
    -   Notes
-   Recent and pinned apps
-   Room database persistence
